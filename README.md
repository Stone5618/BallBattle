# 球球大作战 (BallBattle)

仿"球球大作战"（Agar.io）风格的 Android 游戏，使用 LibGDX 框架开发。

## 功能特性

- 🎮 **基础玩法**: 触摸控制移动、吃食物变大、大球吃小球
- 💥 **分裂机制**: 双击屏幕分裂攻击
- ⚡ **加速机制**: 手指远离屏幕中心加速，消耗质量
- 🤖 **AI对手**: 15-25个AI，5种行为状态（漫游/觅食/追击/逃跑/分裂攻击）
- 🎯 **3种模式**: 自由模式(FFA)、团队模式(红蓝队)、生存模式(地图缩小)
- 🎨 **皮肤系统**: 18种颜色皮肤
- 🏆 **排行榜**: 本地存储历史最高分

## 技术栈

- **框架**: LibGDX 1.12.1
- **语言**: Java 17
- **构建**: Gradle 8.4 + Android Gradle Plugin 8.1
- **目标SDK**: Android 34 (Android 14)
- **最低支持**: Android 5.0 (API 21)

## 安装

下载 `release/球球大作战.apk` 传到手机安装即可。

## 构建

```bash
# 需要 JDK 17+ 和 Android SDK
gradle :android:assembleDebug
```

## 操作说明

| 操作 | 功能 |
|------|------|
| 触摸拖动 | 控制移动方向 |
| 双击 | 分裂 |
| 手指远离中心 | 加速 |
